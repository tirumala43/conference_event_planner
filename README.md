# Conference Expense Planner 🏢💰

A React & Redux-based application to manage and estimate expenses for conferences and events.

## 🚀 Live Demo
[Check it out on GitHub Pages](https://yourusername.github.io/conference-planner/)

## 🛠️ Tech Stack
- React (Vite)
- Redux Toolkit

## 📦 Installation
1. Clone the repo:
-git clone https://github.com/yourusername/conference-planner.git

## 2. Install dependencies:
npm install

## 3. Start the development server:
npm run dev

## 📜 Features
✅ Venue & Add-ons Selection  
✅ Dynamic Cost Calculation  
✅ State Management with Redux
